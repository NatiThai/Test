ไม่ต้องมาพูดพร่ำทำเพลง มารู้จักคำสั่งแรกของภาษา ไพธอน กันเลย คำสั่งที่ว่านี้คือ print เป็นคำสั่งสำหรับแสดงผลข้อความออกทางหน้าจอ:

โค้ด
print ("My first Python code!")
ผลลัพธ์
My first Python code!

นั่นคือคำสั่ง print จะนำข้อความที่อยู่ใน ( ) ออกมาแสดงผลออกทางหน้าจอ โดยข้อความเหล่านั้นจะต้องอยู่ระหว่างเครื่องหมาย " " หรือ ' ' ด้วย
